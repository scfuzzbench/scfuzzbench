# Transparency

This page shows the AWS cost footprint of the public `scfuzzbench` benchmark
account.

- Data source: AWS Cost Explorer
- Metric: `UnblendedCost`
- Currency: `USD`
- Refresh model: generated during the docs build and published as a static page
- Raw data: [`/data/aws-costs.json`](/data/aws-costs.json)

Month-to-date values are estimated and the current UTC day may be partial.
